Reset aplikacji serwerowej
1. Usunięcie danych w bazie danych.
• Aby usunąć dane w bazie danych, należy w terminalu dataplicity wykonać następujące komendy:
o sudo mysql
o DROP DATABASE my_application;
o CREATE DATABASE my_application;

• Następnie należy wyjść z widoku mysql za pomocą CTRL + Z
2. Wykonanie migracji w systemie
• Aby wykonać migracje serwerowe, należy w terminalu dataplicity wykonać następujące komendy:
o cd ~
o . I-dom/bin/activate
o cd I-dom/I-DOM/
o python manage.py makemigrations
o python manage.py migrate
3. Finalizacja resetu danych

• Żeby wprowadzić zmiany w działaniu serwera, należy wykonać następującą komendę:
o sudo service apache2 restart
