IDOM

IDOM is an (engineering project) application for running smart house created with raspberry pi and sensors. Landing page - https://adriannajmrocki.github.io/idom-website/

Play Store

https://play.google.com/store/apps/details?id=com.project.idom

Project status

Project is still in development, full product will be available around January 2021
